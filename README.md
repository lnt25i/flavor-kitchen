# Flavor Kitchen by Chef Raben

Professional food truck website for [flavorkitchen.us](https://flavorkitchen.us).

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Images

Placeholder images use Unsplash URLs. Replace with real photos in:

- `/public/images/hero/`
- `/public/images/menu/`
- `/public/images/chef/`
- `/public/images/gallery/`

Search the codebase for `REPLACE WITH REAL PHOTO` comments when swapping assets.

## Deploy

```bash
npx vercel --yes
```

Custom domain: `flavorkitchen.us`
